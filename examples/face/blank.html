This is an example of how to use the <face> tag in an HTML browser.

Try it by typing in: "Smile"

Here is the category which produces this result:

<category>
<pattern>SMILE</pattern>
<template>
<face value="/images/face/smile.gif"/>
Ok.
</template>
</category>

Some other matching patterns include:

LOOK LEFT
LOOK RIGHT
LOOK RIGHT AND DISPLAY
