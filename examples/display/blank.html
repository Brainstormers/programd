This is an example of how to use the <display> tag in an HTML browser.

Try it by typing in: "Show me Slashdot"

Here is the category which produces this result:

<category>
<pattern>SHOW ME SLASHDOT</pattern>
<template>
Here is where we do a display of some kind.
<display target="target1">http://www.slashdot.org</display>
Can you see it?
</template>
</category>

Some other matching patterns include:

SEARCH GOOGLE FOR *
SEARCH YAHOO FOR *
SEARCH YAHOO AUCTIONS FOR *
SEARCH EBAY FOR *
